Fantasy Premier League Helper

Fantasy Premier League Helper is a Django based app for optimizing youruser Team in Fantasy Premier League game using data from official game API. To learn more about game itself please visit Official Fantasy Premier League site.

Key Notes about apps:

  • App build in App Datta Able- Django Dashboard - Visit;
  • PostgreSQL Database, Django Native ORM;
  • Data from FPL API - List of endpoints;
  • Session-Based Authentication, Forms validation;
  • Deployment scripts: Docker - Repository on GitHub;
